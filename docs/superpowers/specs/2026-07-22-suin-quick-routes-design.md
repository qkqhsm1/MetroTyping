# 수인·분당선 빠른 여행 축소 설계

## 목표

수인·분당선 설정 화면의 빠른 여행을 다음 네 카드로만 제한한다.

- 인천 ↔ 오이도
- 인천 ↔ 왕십리
- 죽전 ↔ 고색
- 인천 ↔ 청량리

각 카드는 기존과 동일하게 양방향 시작 버튼을 제공한다.

## 설계

`Line` 데이터에 선택적인 빠른 여행 역 쌍 목록을 선언한다. `getQuickRoutePairs()`는 이 목록이 있는 노선에서는 선언된 쌍만 기존 `getRoute()`로 계산하고, 목록이 없는 노선에서는 현재처럼 모든 운행 종착역 조합을 자동 생성한다.

UI에 수인·분당선 전용 조건문을 추가하지 않는다. 직접 출발역과 도착역을 고르는 사용자 지정 여행, 역 순서, 운행 종착역 출처 데이터, 다른 노선의 빠른 여행은 변경하지 않는다.

## 검증

- 수인·분당선 빠른 여행 카드가 정확히 네 쌍인지 확인한다.
- 네 쌍이 각각 양방향이며 선택한 출발역과 도착역으로 완전한 경로를 만드는지 확인한다.
- 다른 노선의 기존 자동 조합 생성이 유지되는지 확인한다.
- `npm run check`를 통과한 뒤 `main`에 푸시하고 GitHub Pages 배포를 확인한다.
