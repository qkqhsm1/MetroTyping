# 사운드 선명도 개선 설계

## 목표

정상 재생되지만 너무 작고 둔하게 들리는 현재 효과음을 노트북 스피커에서도 인식하기 쉽게 조정한다.

## 변경

- 전체 피크 게인을 `0.045`에서 `0.09`로 높인다.
- 키 입력음을 `260Hz` 사인파에서 `520Hz` 삼각파로 바꾼다.
- 정답 `660Hz` 사인파, 오답 `125Hz` 톱니파, 완료 `880Hz` 사인파와 80ms 감쇠는 유지한다.
- 음소거, 게임 이벤트 연결, `AudioContext` 종료 동작은 변경하지 않는다.

## 검증

단위 테스트에서 각 효과음의 주파수, 파형, 게인, 감쇠 종료 시점과 음소거를 확인하고 전체 품질 게이트를 실행한다.
